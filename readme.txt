== Description ==

Viavi Dummy Content Generator is simple and easy plugin to generate dummy content.
 
= Features =
  * Easy and simple.
  * Easy setting and configuration
  * Using sortcode  

= Author =
Developed by [viaviweb.com](http://viaviweb.com)
 
== Installation ==

Just flow the following easy steps..

1.Unzip and Upload all files in viavi-dummy-content-generator.zip to the /wp-content/plugins/ directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Click the 'Dummy Content' button at admin menu
